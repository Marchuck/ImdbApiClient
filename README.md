# ImdbApiClient

Welcome to the IMDB API Client for Android, Kotlin project created by [marchuck][github]  with awesome IMDB API^[imdb_api]

## Setup
clone repo, enter api key in your `local.properties` file
```
imdbApiKey=\"k_********\"
```

Feel free to contribute!
